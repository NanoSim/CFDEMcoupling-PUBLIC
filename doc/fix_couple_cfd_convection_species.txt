couple/cfd/speciesConvection command :h3

[Syntax:]

fix ID group-ID couple/cfd/speciesConvection speciesName supersat species0 speciesInitialValue :pre

ID, group-ID are documented in "fix"_fix.html command :ulb,l
couple/cfd/convection = style name of this fix command :l
speciesName = obligatory keyword :l
supersat = name of the species to be coupled (will generate a fix property/atom with this name) :l
species0 = obligatory keyword :l
speciesInitialValue = Initial value for the per-particle species :ls
:ule

[Examples:]

fix speciesConv1 all couple/cfd/speciesConvection speciesName supersat species0 1.01

[Description:]

This fix is responsible for the coupling species quantities between CFD and DEM calculation, i.e. for pushing and pulling of per-particle properties that are necessary to calculate the interphase species fluxes.

This fix handles explicit and implicit flux coupling automatically. Thus, ALL relevant exchange fields for the species, including transfer coefficients (name '*TransCoeff) and fluid quantities (name '*Fluid') are generated by this fix.

[Restrictions:]

None.

[Related Commands:]
"fix couple/cfd/force"_fix_couple_cfd_force.html

[Default:]

None
